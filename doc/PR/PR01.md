# PR#1: プロジェクト初期設定

## 概要

Next.js 16プロジェクトの作成と開発環境の構築を行いました。

## 実装内容

### 1. Next.js プロジェクト作成

- Next.js 16.0.10（App Router）
- React 19.2.1
- TypeScript 5.x

### 2. TypeScript 設定

- `tsconfig.json` - strictモード有効、パスエイリアス `@/*` 設定

### 3. Tailwind CSS 設定

- Tailwind CSS 4.x
- `postcss.config.mjs` - @tailwindcss/postcss プラグイン
- カスタムカラー（LINE green、primary）
- 日本語フォントファミリー設定

### 4. ESLint 設定

- `eslint.config.mjs` - ESLint 9.x flat config形式
- next/core-web-vitals, next/typescript 拡張
- カスタムルール追加

### 5. Prettier 設定

- `.prettierrc` - セミコロン、シングルクォート、Tailwindプラグイン
- `.prettierignore` - 除外パターン

### 6. ディレクトリ構造

```
src/
├── app/
│   ├── layout.tsx
│   ├── page.tsx
│   └── globals.css
├── components/
│   ├── common/
│   ├── upload/
│   ├── result/
│   └── animation/
├── hooks/
├── lib/
└── types/

tests/
├── unit/
├── integration/
└── e2e/
```

### 7. Jest 設定

- `jest.config.ts` - Next.js統合、jsdom環境
- `jest.setup.ts` - @testing-library/jest-dom、グローバルモック

### 8. Playwright 設定

- `playwright.config.ts` - Chrome、Mobile Safari、Mobile Chrome
- 自動開発サーバー起動、スクリーンショット・トレース設定

### 9. CI/CD 設定

- `.github/workflows/ci.yml`
  - lint: ESLint、Prettier
  - type-check: TypeScript
  - test: Jest（カバレッジ付き）
  - build: 本番ビルド
  - e2e: Playwright E2Eテスト

### 10. VSCode 設定

- `.vscode/settings.json` - 保存時フォーマット、ESLint自動修正
- `.vscode/extensions.json` - 推奨拡張機能

### 11. その他

- `.gitignore` - node_modules、.next、coverage等
- `.env.example` / `.env.local` - API_BASE_URL設定
- `README.md` - セットアップ手順、コマンド一覧
